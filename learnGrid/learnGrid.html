CSS Grids

The purpose of today's study is to explore CSS Grid Layouts.

References: MDN Web Docs

The first step to working with CSS grid layouts is creating a wrapper class.

.wrapper { border-style: solid; border-width: 1px; display: grid; grid-template-columns: repeat(3, 1fr); gap: 10px; grid-auto-rows: minmax(100px, auto) }

The perimeter of the wrapper class is shown with a black border.

Dividing the wrapper class into three rows and three colums yields nine work areas.

I labeled my classes according to the following schema:

w{width}h{height}atr{row}c{column}

where row and column represent the upper left corner of the space.

/* Column 1 */ .w1h1atr1c1 { background-color: cadetblue; grid-column: 1; grid-row: 1; } .w1h1atr2c1 { background-color: cadetblue; grid-column: 1; grid-row: 2; } .w1h1atr3c1 { background-color: cadetblue; grid-column: 1; grid-row: 3; } /* Column 2 */ .w1h1atr1c2 { background-color: cadetblue; grid-column: 2; grid-row: 1; } ...etc.

Height: 1

Width: 1

There are six possible areas where the width to height ratio is 2:1.

/* Column 1 */ .w2h1atr1c1 { background-color: greenyellow; grid-column: 1/3; grid-row: 1; } .w2h1atr2c1 { background-color: greenyellow; grid-column: 1/3; grid-row: 2; } ... /* Column 2 */ .w2h1atr1c2 { background-color: greenyellow; grid-column: 2/4; grid-row: 1; } ...

Height: 1

Width: 2

There are also six possible positions for width to height ratio is 1:2.

/* Row 1 2*/ .w1h2atr1c1 { background-color: palevioletred; grid-column: 1; grid-row: 1/3; } .w1h2atr1c2 { background-color: palevioletred; grid-column: 2; grid-row: 1/3; } ... /* Row 2 */ .w1h2atr2c1 { background-color: palevioletred; grid-column: 1; grid-row: 2/4; } ...

Height: 2

Width: 1

Since the wrapper class is using auto to generate rows, if nothing is created in a row, the row won't be created at all. For example, the following code is supposed to create a width to height ratio of 2:1 instead of 3:1, but the third row does not get created.

<div class="wrapper"> <div class="w1h2atr1c1"></div> </div>

Other examples of dividing a nine by nine grid.

.w2h2atr1c1 { background-color: yellow; grid-column: 1/3; grid-row: 1/3; } ... .w2h2atr2c2 { background-color: yellow; grid-column: 2/4; grid-row: 2/4; } ...

Width: 2
